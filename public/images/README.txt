========================================
HOW TO ADD PHOTOS TO YOUR WEBSITE
========================================

STEP 1: Put your photos in this folder
   - Supported formats: JPG, PNG, WebP
   - Recommended size: 800x1200 pixels or similar
   - Name them clearly: photo1.jpg, photo2.jpg, etc.

STEP 2: Update the HTML file
   - Open index.html in a text editor
   - Find the Photo Reel Section (search for "gallery")
   - Replace the image paths like this:
     OLD: images/photo1.jpg
     NEW: images/your-actual-photo-name.jpg

STEP 3: Refresh your browser
   - Save the HTML file
   - Refresh your browser (Ctrl+F5)

EXAMPLE:
If you have a photo named "award-ceremony.jpg" in this folder,
change: src="images/photo1.jpg"
to:     src="images/award-ceremony.jpg"
